Deployment steps with Heroku

    Step 1: Please create signup on the below portal and login on the portal.
            https://signup.heroku.com/
    Step 2: Install heroku CLI from heroku portal.(Will display after login on the portal)
    Step 3: Check heroku install successfully or not from the command
            Run ->cmd ->heroku login
    Step 4: First create a sample heroku project using below command.
            git clone https://github.com/heroku/node-js-getting-started.git
            cd node-js-getting-started
    Step 5: Create and deploy the app using below command
            heroku create
            git push heroku main
    Step 6: Now you can go to the browser and hopefully your application run (please check url on the console)


/* Pushing the new code into Heroku Portal and inside the main branch */

Perform below command..below command is used for commit the new change and deploy the changes.
git add . 
git commit -m "Added dynamic port for the application"
git push heroku main


For any error, please check with below command
heroku logs --tail

NOTE: Need to changes in package.json and changes the port for run the application.
app.listen(process.env.PORT || 3000, function(){
  console.log("Express server listening on port %d in %s mode", this.address().port, app.settings.env);
});
